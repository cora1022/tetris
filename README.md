# tetris

브라우저에서 테트리스 로직을 실험하는 초기 HTML/JavaScript 프로토타입입니다.

## 프로젝트 목적

2차원 배열로 게임 보드를 표현하고, 블록 위치/충돌/하강 로직을 직접 구현해 보는 학습용 실험입니다. 현재는 완성된 게임보다 로직 검증용 prototype에 가깝습니다.

## 기술 스택

- HTML
- Vanilla JavaScript

## 아키텍처

`tetris.html` 파일 내부에 보드 배열, 블록 데이터, 충돌 검사, 블록 하강 함수가 함께 정의되어 있습니다. 렌더링과 게임 루프는 아직 완성 단계가 아니며, 콘솔 로그를 통해 상태를 확인하는 구조입니다.

## 폴더 구조

```text
.
├── tetris.html
└── README.md
```

## 실행 방법

브라우저에서 `tetris.html`을 열고 개발자 도구 콘솔을 확인합니다.

```bash
python -m http.server 8000
```
